## Conventions

- Filename: `ADR-NNN-<kebab-slug>.md` (zero-padded NNN)
- Status: `Proposed` | `Accepted` | `Superseded by ADR-MMM`
- Never delete an ADR — supersede instead (extended to handoffs + retros per ADR-011)
- **NNN allocation**: reserve the row in this INDEX *before* authoring the ADR file (per ADR-009). Post-hoc renumber is the fallback, not the primary protocol.
- Format prescribed by `~/Projects/cascade-system/docs/rules/document-as-you-go.md`
